### 目录结构

```
├─ api              封装 httpclient，调用其他服务接口
├─ build            构建、部署脚本
├─ configs          配置文件
├─ controller       处理用户输入
├─ cronjob          定时任务
├─ database         数据库相关逻辑
├─ errors           错误处理封装
├─ logger           日志
├─ middlewares      中间件
├─ model            数据模型
├─ routers          路由
├─ service          业务逻辑
└─ tools            工具
├─ main.go          
├─ go.mod           
├─ README.md
```
